 * 로브 > Robes/Custome Garmnents

 * 악세사리 > Headgears
    * /여 = Female
    * /남 = Male
 * 방패 > Shields
    * /Class = Folders for Shields per Class
 * 인간족 > Class Sprites
    * /Class = Folders for Weapons and attack animations per Class
    * /머리통 = Head Sprites

        * /남 = Male
   * /몸통 = Body Sprites

        * /남 = Male  
        Files: /Class_Gender.act/spr (See at the end for complete list)
 * 이팩트 > Skill and Hat Effects
 * npc > NPC Sprites
